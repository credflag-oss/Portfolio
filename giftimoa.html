🎁 기프티모아

가족과 함께 쓰는 기프티콘 관리 앱
Android · Kakao Login · Firebase · ML Kit OCR

프로젝트 개요

기프티콘을 카메라로 촬영하면 ML Kit OCR이 자동으로 브랜드명·상품명·바코드·유효기간을 인식하고, 가족 구성원과 공유·관리할 수 있는 Android 앱입니다.

플랫폼

Android (Kotlin + Jetpack Compose)

상태

주요 개발 진행 중

Clean Architecture + MVVM

인증

Kakao Login SDK → Firebase Custom Auth

Kotlin Jetpack Compose Firebase Firestore Firebase FCM Firebase Storage Kakao Login ML Kit OCR CameraX ZXing Hilt DI Clean Architecture

데이터 모델 설계서

아키텍처

Clean Architecture 3-Layer + MVVM 패턴으로 구성됩니다.

Presentation Layer

Jetpack Compose UI · ViewModel · StateFlow · UiState sealed class

Domain Layer

UseCase · Repository Interface · Entity · 순수 비즈니스 로직

Data Layer

Firebase Firestore · Storage · FCM · Kakao SDK · ML Kit

DI

Hilt — Module 단위 의존성 주입 (Network / Firebase / Repository)

주요 기능

📷 OCR 자동 인식

CameraX + ML Kit Text Recognition으로 기프티콘 촬영 시 브랜드·상품명·바코드·유효기간 자동 추출

👨‍👩‍👧 가족 공유

6자리 초대 코드로 가족 그룹 생성. ADMIN / MEMBER / CHILD 역할별 권한 분리

🔔 만료 알림

Cloud Scheduler + FCM으로 D-7 / D-3 / D-1 만료 임박 푸시 알림 자동 발송

🔒 바코드 보안

바코드 번호 AES-256 암호화 저장. 복호화 키는 Android Keystore 보관

📴 오프라인 지원

Firestore 로컬 캐시로 네트워크 없이도 기프티콘 바코드 조회 가능

🖼️ 이미지 최적화

Cloud Functions 자동 썸네일 생성. 목록은 썸네일, 상세만 원본 로드
